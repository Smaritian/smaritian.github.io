初识Java

Java分类

  • JavaSE:基础
  • JavaEE:企业服务端开发
  • JavaME：移动设备开发

Java环境搭建

  1. jdk（java development kit）
    1. JVM（java virtual machine，java虚拟机）
    2. 核心类库
    3. JRE（java runtime environment）
  2. 系统变量JAVA_HOME, 环境变量%JAVA_HOME%\bin
  3. cmd：java，验证安装及配置是否成功

Java执行流程

  1. 编写源代码，java文件
  2. cmd：javac，编译java文件为class文件（二进制）
  3. cmd：java，执行class文件
